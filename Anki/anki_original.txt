Como retornar um sequência aleatória de x bytes no formato de bytes com o random? random.randbytes(x)
Para que serve a função random.getstate()? Retorna uma tupla de dados que foram utilizados para gerar o número aleatório naquele momento.
Para que serve a função random.setstate(state)? Utiliza os dados de random.getstate() para definir uma semente.
Como criar uma semente para valores aleatórios na biblioteca random? random.seed() <br><br> Só funciona para a célula atual.
Como misturar os elementos de uma lista com o random? random.shuffle(lista)
Como gerar valores aleatórios inteiros com o random? random.randint(valor mínimo, valor máximo inclusive) <br><br> random.randrange(valor mínimo, valor máximo inclusive, passo)
Como escolher um elemento aleatório de uma lista usando o random? random.choice(lista)
Como escolher uma lista de tamanho n, com elementos aleatório de outra lista com reposição? random.choices(population = lista, k = n) <br><br> weights: É uma lista de mesmo tamanho que a lista base, indicando o peso de cada elemento para ser selecionado
Como escolher uma lista de tamanho n, com elementos aleatório de outra lista sem reposição? random.sample(population = lista, k = n)
Como gerar valores aleatórios entre 0 e 1 usando random? random.random()
